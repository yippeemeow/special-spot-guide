# مشروع مسارك - Masarak 📍

## الوصف
"مسارك" هو منصة ذكية للملاحة الداخلية (Indoor Navigation) مصممة لرفع كفاءة التدفق اللوجستي في الفعاليات والمعارض الكبرى. يركز التطبيق على تجربة المستخدم الشاملة، مع توفير أدوات متقدمة لإمكانية الوصول (Accessibility) لضمان تنقل سلس لجميع الزوار، بما في ذلك ذوي الاحتياجات الخاصة.

## التقنيات المستخدمة (Stack)
بناءً على البنية التقنية للمشروع، تم استخدام أحدث المكتبات لضمان السرعة والاستقرار:
* **Frontend Framework:** React 18.3 مع Vite (لأداء فائق السرعة).
* **Language:** TypeScript (لضمان جودة الكود وتقليل الأخطاء).
* **Database & Backend:** Supabase (لإدارة البيانات والمواقع بشكل لحظي).
* **UI Components:** Radix UI و Shadcn UI (لتوفير واجهات وصول قياسية).
* **Styling:** Tailwind CSS مع Tailwind-animate (لتصميم عصري وتفاعلي).
* **State Management:** TanStack React Query v5 (لإدارة جلب البيانات بكفاءة).
* **Data Visualization:** Recharts (لتحليل وعرض بيانات تدفق الزوار).
* **Validation:** Zod مع React Hook Form (لضمان دقة البيانات المدخلة).

## المزايا اللوجستية
* **تقليل الهدر:** تحسين مسارات المشي لتقليل Motion Waste.
* **دقة البيانات:** ربط مباشر مع قاعدة بيانات Supabase لتحديث مواقع البوثات فورياً.
* **الشمولية:** استخدام مكونات Radix UI التي تدعم قارئات الشاشة والتنقل بلوحة المفاتيح.
